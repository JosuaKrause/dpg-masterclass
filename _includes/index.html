What are Digital Public Goods?

Digital Public Goods (DPGs) are open-source software, open data, open AI models, open standards, and open content that adhere to privacy and other applicable laws and best practices, do no harm, and help attain the Sustainable Development Goals (SDGs). They are digital, can be infinitely stored, copied, and distributed without becoming depleted, and at close to zero cost.

GitHub is a platform that fosters open-source development, making it an ideal tool for working on DPGs. It allows for seamless collaboration between different contributors, and provides tools for version control, issue tracking, and code review.

DPG Masterclass Hands-on Activities

The hands-on activities for this masterclass are mainly focused on two key aspects of using GitHub:

  • Documenting: We’ll guide you through the process of creating and maintaining comprehensive documentation for your DPG projects. Good documentation is key to the success of any project, and we’ll show you how to do it right on GitHub.
  • Collaborating: GitHub is all about collaboration. We’ll show you how to use issues, pull requests, and other GitHub features to collaborate effectively with your team and the wider GitHub community.
